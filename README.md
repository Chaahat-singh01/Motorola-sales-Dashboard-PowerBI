# 📊 Motorola Sales Dashboard | Power BI

## 📌 Project Overview
This interactive Power BI dashboard analyzes Motorola sales performance to uncover trends in revenue, sales quantity, customer ratings, payment methods, and regional performance.

## 🎯 Business Objective
The goal of this project is to help business stakeholders monitor sales performance, identify top-performing products and cities, and support data-driven decision-making.

## 🛠️ Tools Used
- Power BI
- Power Query
- DAX
- Excel

## 📈 Key KPIs
- Total Sales
- Total Quantity Sold
- Total Transactions
- Average Sales

## 📊 Dashboard Insights
- Sales performance across different cities
- Monthly sales trend analysis
- Top-selling mobile models
- Customer rating distribution
- Payment method analysis
- Sales performance by day of the week
